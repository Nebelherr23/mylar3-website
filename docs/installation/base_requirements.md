# Base requirements

Mylar3 requires:

- a computer
- [Python](https://www.python.org)
    - Minimum version: **3.8.1**
    - Recommended version: **3.11.4**
- unrar (linux/macOS/Windows) or winrar (windows)
- an API key from [ComicVine](https://comicvine.gamespot.com/api)
